ChemoSens tools - Statistical analyses

How it works?

1) Download the XLSX template associated with the chosen analysis by clicking below:

Time variability plots

2) Replace the data in the "Data" sheet of the XLSX template.

3) Change the default parameters in the "Parameters" sheet of the XLSX template (optional).

4) Save the template without renaming it.

5) Click on "Run", select the updated template, then wait for the analysis to be run on the server.

6) Visualize the results on an HTML page. Click on a picture to zoom, then right-click on "save picture as" to save it or click on "Download as ZIP"/"Download as PPTX" to download all the outputs as an archive (ZIP) or PowerPoint (PPXT) file.
