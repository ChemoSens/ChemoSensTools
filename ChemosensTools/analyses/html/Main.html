ChemoSens tools - Statistical analyses

How it works?

1) Download the template associated with the chosen analysis.

2) Replace the data in the "Data" tab.

3) Change the parameters in the "Parameters" tab (requires a paid account).

4) Click on "Run", wait for the analysis to be run on the server, then visualize the results on an HTML page.

5) Optionally click on a picture to download it, or on "Download as ZIP"/"Download as PPTX" to download all the outputs as an archive (ZIP) or PowerPoint (PPXT) file.
