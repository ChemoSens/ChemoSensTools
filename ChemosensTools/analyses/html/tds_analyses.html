TDS statistical analyses

How it works?

1) Download the XLSX template associated with the chosen analysis by clicking below:

TDS template

2) Replace the data in the "Data" sheet of the XLSX template.

3) Change the parameters in the "Parameters" tab (optional).

4) Click on "Run", wait for the analysis to be run on the server, then visualize the results on an HTML page.

5) Click on a picture to zoom, then right-click on "save picture as" to save it or click on "Download as ZIP"/"Download as PPTX" to download all the outputs as an archive (ZIP) or PowerPoint (PPXT) file.
