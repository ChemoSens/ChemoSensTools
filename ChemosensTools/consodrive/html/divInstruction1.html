Vos critères de choix alimentaires

Pour chaque aliment de votre panier (facture), nous allons vous demander de préciser les caractéristiques qui ont été déterminantes lors de votre décision d'achat.

Les caractéristiques susceptibles de guider vos choix étant très nombreuses, nous les avons rassemblées en 8 grandes familles afin de faciliter vos réponses.

Ces 8 grandes familles de raisons d'achat sont décrites dans les pages suivantes de ce questionnaire. Nous vous demandons de lire attentivement ces descriptions ainsi que les exemples qui illustrent les différentes familles.

Vous réaliserez ensuite un exercice consistant à reclasser les exemples cités précédemment dans les bonnes familles. Prenez cet exercice comme un jeu qui n'a pour objectif que de vous familiariser avec ces familles et de faciliter vos réponses par la suite.

Cliquez sur "Suivant" pour commencer à lire les descriptions.
