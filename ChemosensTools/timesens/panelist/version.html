2.0.1

Date: 2017/09/19

TimeSens is now a Progressive Web App (read more on Progressive Wep Apps here):

- Support for offline mode [BETA] (Chrome and Firefox browsers only): once started online, TimeSens is downloaded in the browser cache an can be reused offline.

- Support for application manifest on mobile devices (installable Web App) [BETA]: TimeSens can now be added to the home screen on a phone or tablet, and can be run full screen and separate of the browser.

- Check for browser compatibility.

- New interface, more intuitive.

   - TDS with no dominance periods (fading)
